This project is pre-configured to work with GitHub Pages.

### Steps to Deploy:

1.  **Push to GitHub:**
    Ensure this directory is initialized as a git repository and pushed to GitHub.
    ```bash
    git init
    git add .
    git commit -m "Initial commit"
    git branch -M main
    git remote add origin https://github.com/YOUR_USERNAME/REPO_NAME.git
    git push -u origin main
    ```

2.  **Activate Pages:**
    - Go to your repository on GitHub.
    - Navigate to **Settings** > **Pages**.
    - Under **Build and deployment** > **Source**, select **Deploy from a branch**.
    - Under **Branch**, select `main` and `/ (root)`.
    - Click **Save**.

3.  **Verify:**
    Wait a minute or two, and your site will be live at the URL provided in the settings!
